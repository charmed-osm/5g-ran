# Integration tests

Functional tests for 5G RAN were created using zaza.

## Install tox

```bash
apt-get install tox
```

## Run Integration Test

To run zaza integration test,

```bash
tox -e func_test
```

This command will create a model, deploy the charms, run tests and destroy the
model.
